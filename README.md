# Project CLI Python

## Deskripsi
Aplikasi CLI sederhana berbasis Python yang bertemakan sistem login dan register (Pilihan tema ke-2) dan penerapan nyata pada sistem perpustakaan sederhana.

## Fitur
- Register akun
- Login akun
- Melihat katalog buku
- Meminjam buku
- Melihat daftar pinjaman
- Mengembalikan buku

## Teknologi
- Python 3.14.3

## Cara Menjalankan
```bash
python main.py
```
